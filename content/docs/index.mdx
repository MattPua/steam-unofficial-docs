## Introduction

The Steam Web API is a RESTful API that allows you to interact with the Steam platform.

For more information please see [Steam API](https://steamcommunity.com/dev) page or the [Valve Developer Wiki](https://developer.valvesoftware.com/wiki/Steam_Web_API).

## Obtaining API Key

You can obtain an API key [here](https://steamcommunity.com/dev/apikey).
